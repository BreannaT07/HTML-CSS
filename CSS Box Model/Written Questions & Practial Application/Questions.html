Written Response Questions

Define the CSS Box Model in your words.
The CSS Box Model is a model that can wrap around every HTML element put into it, like box-sizing, padding, margin properties, etc.


What are the four main components of the CSS Box Model? Describe each one briefly.
Padding: Creates generated spaces around an element's content, inside of any kind of defined border.

Margin: Creates space around elements, outside any type of defined border.

Border: Allows to specify the style, width, and color of the element's border, any type that is defined inside and outside.

Box-Sizing: Allows to include in the element's padding and border the total width and height.


Explain the difference between "content-box" and 'border-box" sizing models.
The "content-box" adds margin, padding, and borders outside of the box, while a "boder-box" adds margin, padding, and borders inside of the box.


How does "margin" affect an element's positioning on a webpage?
A margin affects an element's positioning on a webpage as they are the added space in the box.


Why is "box-sizing: border-box;" often recommened when working with layout design?
It is often recommened because it is ideal for making responsive websites.
